## front-end

### webpack

_`webpack`是静态资源模块化打包工具。_

**webpack 打包代码依赖核心内容**

```
yarn add webpack@4.46.0 --dev
```

**webpack 命令行工具**

```
yarn add webpack-cli@4.10.0 --dev
```

**命令行使用方式**

```
npx webpack --entry ./src/main.js --output-path ./build
```
